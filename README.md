## Description

Simple SSE example for [Nest](https://github.com/nestjs/nest).
As SSE now is part of NestJs better follow the [official documentation for SSE](https://https://docs.nestjs.com/techniques/server-sent-events). 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
